# Kata-Losange

## Kata Description
Ecrire en programme qui : 

qui prend en entrée une lettre, affiche un losange commencant par 'A', et qui possède la lettre en entrée sur sa diagonale centrale. 

![Alt text](/kata.png?raw=true "Optional Title")

### Examples
```
Input: A
Output: A

Input: B
Output:   A
         B B
          A


Input: D
Output:    A
          B B
         C   C
	    D     D
         C   C
          B B
           A
```
